The background of this page is a radial gradient. The table below shows some other types of gradients as well as the HTML5 code required for creating them.

Types of Gradients
Type of Gradient HTML5 Code Example
Linear Gradient: Left to Right background: linear-gradient(to right, black, #00ccff, white)
Linear Gradient: Top to Bottom background: linear-gradient(#ffff00, #ff6600, #ff0000)
Linear Gradient: Diagonal background: linear-gradient(to bottom right, #00ff00, #1affb2, #e6ccff)
Linear Gradient: Transparency background: linear-gradient(to right, rgba(254,129,255,0), rgba(254,129,255,1))
Radial Gradient background: radial-gradient(circle, #7300e6, #ff4dff, #9494b8)
