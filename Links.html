There are some properties which you can apply to links to change their appearances depending on whether they were clicked or not or hovered over.

a:link{
color: cyan;
background-color: transparent;
text-decoration: none;
}		 a:visited{
color: pink;
background-color: transparent;
text-decoration: none;
}		 a:hover{
color: lime;
background-color: transparent;
text-decoration: underline;
}		 a:active{
color: fuchsia;
background-color: transparent;
text-decoration: underline;
}
This changes the default color(blue) of an unclicked link to cyan and removes the underline. A default visited link is purple and underlined. This makes visted links pink without the underline. As you hover over a link, it will change from its default color or set color to lime-green and will be underlined. As you are clicking on a link, it will appear fuchsia and underlined.
